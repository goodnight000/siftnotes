# SiftNotes Updates

SiftNotes uses the Tauri updater plugin. One manual bootstrap install is still required after updater support changes, because the already-installed app was built with updater artifacts and endpoints disabled.

After that bootstrap install, future releases can update in-app from:

`https://github.com/charleszheng/siftnotes/releases/latest/download/latest.json`

Before the first update-enabled build, change `frontend/src-tauri/tauri.conf.json` if you want a different static host, GitHub repo, S3 bucket, R2 bucket, or domain.

The checked-in updater public key must match the private key used for release builds. If you do not have the matching private key for the current `plugins.updater.pubkey`, generate a new key pair and replace the public key before making the bootstrap build.

## Release Checklist

1. Keep the updater signing private key outside git.
2. Generate a password-protected updater key if you do not already have one:

```sh
pnpm --dir frontend tauri signer generate --write-keys ~/.siftnotes/updater.key
```

3. Copy the generated public key into `frontend/src-tauri/tauri.conf.json` under `plugins.updater.pubkey`.
4. Build with updater signing environment variables set:

```sh
TAURI_SIGNING_PRIVATE_KEY_PATH="$HOME/.siftnotes/updater.key" \
TAURI_SIGNING_PRIVATE_KEY_PASSWORD='...' \
pnpm --dir frontend tauri:build
```

5. Upload the generated installer/update artifact, `.sig`, and `latest.json` manifest to the configured update endpoint.
6. Bump `frontend/src-tauri/tauri.conf.json` and `frontend/package.json` versions before the next release. Tauri compares the installed app version to the manifest version.

The manifest must include the target platform URL and signature generated by Tauri, for example:

```json
{
  "version": "0.4.1",
  "notes": "SiftNotes update",
  "pub_date": "2026-06-21T00:00:00Z",
  "platforms": {
    "darwin-aarch64": {
      "url": "https://github.com/charleszheng/siftnotes/releases/download/v0.4.1/SiftNotes_0.4.1_aarch64.app.tar.gz",
      "signature": "PASTE_GENERATED_SIGNATURE"
    }
  }
}
```
